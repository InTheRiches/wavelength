# Wavelength

**Wavelength** is a static, information-focused website I built to explore and learn **Next.js**. The site has sidebar navigation with multiple pages of structured content, page routing, and a responsive design.

---

## Purpose

This project was created as a personal learning tool to:

- Gain practical experience with **Next.js** and React frameworks.
- Build a **multi-page, statically generated website**.
- Implement a **sidebar-based navigation system** for content-rich sites.
- Understand file-based routing, component reuse, and static site optimization.
- Learn Tailwindcss

---

## Tech Stack

- **Framework:** Next.js  
- **Frontend:** React, HTML, TailwindCSS
- **Deployment:** Vercel
